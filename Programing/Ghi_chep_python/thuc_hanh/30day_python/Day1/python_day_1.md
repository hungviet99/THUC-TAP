# Bài tập ngày 1

## Mức độ 1

- Kiểm tra phiên bản python bạn đang sử dụng
```
python -V
python
python3 -V
python3 
```
### Mở trình tương tác python và thực hiện các thao tác sau. Toán hạng là 3 và 4. Kiểm tra ví dụ trên
- Phép cộng (+)
- phép trừ (-)
- phép nhân(*)
- lấy dư (%)
- chia (/)
- số mũ(**)
- Phép chia tầng (//)

- Tên của bạn
